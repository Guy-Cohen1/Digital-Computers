main.c - Contains the main finite state machine (FSM) logic.
api.c - Implements high-level application functions.
halGPIO.c - Provides hardware abstraction layer (HAL) functions at a low level, supporting the application's functionality.
flash.c - Manages operations related to flash memory.
bsp.c - Handles the initialization routines for the module.
All header files include declarations of external functions and variables.
GUI.py
GUI.py is a Python script designed to facilitate communication between the MSP430 microcontroller and a PC. The PC controls the microcontroller's operations through a graphical user interface (GUI) with interactive control buttons.

Required Libraries:
PySimpleGUI
time
serial
tkinter
mouse
os
threading
binascii
